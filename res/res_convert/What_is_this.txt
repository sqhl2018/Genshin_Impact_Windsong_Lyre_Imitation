这是我把资源文件打包进exe的工具，目标是把资源文件转成res.o,并获得可用来访问资源的res.h,res.cpp
①新建文件夹res_bin,res_cpp,res_objtmp,res_out
②复制所有用到的资源文件到res_bin
③运行resMake.exe
④在本目录启动命令"make -f res.makefile"
⑤进入res_out目录,复制res.o到res目录,复制res.h、res/cpp到src

This is the tool I use to package resource files into exe. The goal is to convert resource files into res.o and get res.h,res.cpp that can be used to access resources
① Create folders: res_bin, res_cpp,res_objtmp, res_out
② Copy all used resource files to res_bin
③ run resMake.exe
④ Run the "make -f res.makefile" command in this directory.
⑤ step into res_out directory,and copy res.o to the res directory ,res.h and res.cpp to src directory
